# 3.1.4: Implement `cases_assign_attorney` tool

**Status:** BACKLOG
**Type:** Story
**Feature:** [3.1: Case Management Tools](./3.1-case-management-tools.md)
**Tags:** `+case-mgmt`

---

As a partner,
I want to assign or reassign attorneys to cases,
So that case teams are properly staffed.

- [ ] Parameters: `caseId`, `attorneyId`, `role` (Lead/Supporting/Reviewer)
- [ ] Partner-only authorization
- [ ] Validate attorney exists and IsActive = true
- [ ] Create CaseEvent of type Assignment
- [ ] Write to audit log

## Acceptance Criteria

- [ ] Non-partner callers receive 403
- [ ] Assigning an inactive attorney is rejected
- [ ] A CaseEvent is created for every assignment or reassignment
