# 3.5.1: Implement `calendar_get_hearings` tool

**Status:** BACKLOG
**Type:** Story
**Feature:** [3.5: Court Calendar & Deadline Tools](./3.5-calendar-tools.md)
**Tags:** `+calendar`

---

As an attorney,
I want to see upcoming court hearings for my cases or schedule,
So that I can prepare for court appearances in advance.

- [ ] Parameters: `caseId?`, `attorneyId?`, `dateFrom?`, `dateTo?`
- [ ] Return hearings with court name, judge, hearing type, location, and case details
- [ ] Role-based: attorneys see their own; partners see practice group; legal assistants see assigned attorney's

## Acceptance Criteria

- [ ] At least one filter (caseId or attorneyId) should be specified; warn but don't fail if neither is
- [ ] Results include court name, judge contact, and hearing type
- [ ] Results are ordered by DateTime ascending
