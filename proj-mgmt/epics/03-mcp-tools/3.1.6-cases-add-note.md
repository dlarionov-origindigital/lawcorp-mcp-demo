# 3.1.6: Implement `cases_add_note` tool

**Status:** BACKLOG
**Type:** Story
**Feature:** [3.1: Case Management Tools](./3.1-case-management-tools.md)
**Tags:** `+case-mgmt`

---

As an attorney or paralegal,
I want to add notes to a case,
So that observations and updates are captured in the case record.

- [ ] Parameters: `caseId`, `content`, `isPrivileged?`
- [ ] Privileged notes visible only to attorneys (not paralegals or interns)
- [ ] Create CaseEvent of type Note
- [ ] Write to audit log

## Acceptance Criteria

- [ ] Note is stored as a CaseEvent with correct EventType
- [ ] Privileged notes do not appear for paralegal or intern callers
- [ ] Paralegals can add non-privileged notes; interns cannot add notes
