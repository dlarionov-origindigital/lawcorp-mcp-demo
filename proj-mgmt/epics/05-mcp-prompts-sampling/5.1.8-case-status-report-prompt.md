# 5.1.8: `generate_case_status_report` prompt

**Status:** BACKLOG
**Type:** Story
**Feature:** [5.1: Prompt Templates](./5.1-prompt-templates.md)
**Tags:** `+prompts` `+case-mgmt`

---

As an attorney,
I want a prompt that creates a client-facing case status report,
So that I can keep clients informed with a professional update.

- [ ] Arguments: `caseId`, `reportingPeriod`
- [ ] Embed case status, recent timeline events, upcoming hearings and deadlines
- [ ] Tone is professional and client-appropriate (no privileged attorney notes)

## Acceptance Criteria

- [ ] Report covers the specified reporting period
- [ ] Privileged content is excluded from client-facing output
- [ ] Upcoming deadlines within 30 days are highlighted
