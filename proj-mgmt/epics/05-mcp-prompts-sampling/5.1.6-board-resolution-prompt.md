# 5.1.6: `prepare_board_resolution` prompt

**Status:** BACKLOG
**Type:** Story
**Feature:** [5.1: Prompt Templates](./5.1-prompt-templates.md)
**Tags:** `+prompts` `+case-mgmt`

---

As a corporate attorney,
I want a prompt that drafts a board resolution for a corporate action,
So that governance documentation is produced quickly and correctly formatted.

- [ ] Arguments: `caseId`, `resolutionType`, `details`
- [ ] resolutionType drives the recitals and resolved clauses
- [ ] Embed company name and case context

## Acceptance Criteria

- [ ] Output follows standard Delaware board resolution format
- [ ] Resolved clauses are specific to the resolutionType provided
