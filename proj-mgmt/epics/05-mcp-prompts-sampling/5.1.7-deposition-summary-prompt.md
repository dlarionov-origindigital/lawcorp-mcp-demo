**Status:** BACKLOG

As a litigation attorney,
I want a prompt that summarizes a deposition transcript with key testimony highlighted,
So that I can quickly review and prepare for examination.

- [ ] Arguments: `documentId`, `focusTopics?`
- [ ] Fetch document content and embed
- [ ] focusTopics guides which testimony areas to highlight

## Acceptance Criteria

- [ ] Output includes witness name, key admissions, and contradictions with prior statements
- [ ] focusTopics narrows the summary to relevant testimony sections
