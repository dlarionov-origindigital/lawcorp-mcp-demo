# 8.1.2.3: Sign-Out Flow

**Status:** DONE
**Type:** Task
**Story:** [8.1.2: Entra ID OIDC Sign-In](./8.1.2-entra-id-oidc-auth.md)

## Description

Implement sign-out: user clicks "Sign out" → local session cleared → redirect to Entra ID sign-out → redirect back to the app.

## Implementation

- `LoginDisplay.razor` renders a "Sign out" link when authenticated
- The link navigates to `/MicrosoftIdentity/Account/SignOut` (provided by `Microsoft.Identity.Web.UI`)
- The controller clears the local authentication cookie and redirects to Entra ID's `end_session_endpoint`
- After Entra ID sign-out completes, the user is redirected to `/signout-callback-oidc` and back to the app
- `SignedOutCallbackPath` is configured in `appsettings.json` → `AzureAd:SignedOutCallbackPath`

## Key Files

- `src/LawCorp.Mcp.Web/Components/Layout/LoginDisplay.razor`
- `src/LawCorp.Mcp.Web/appsettings.json` (`AzureAd:SignedOutCallbackPath`)
