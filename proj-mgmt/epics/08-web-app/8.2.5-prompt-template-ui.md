# 8.2.5: Prompt template UI

**Status:** BACKLOG
**Type:** Story
**Feature:** [8.2: MCP Client Integration](./8.2-mcp-client-integration.md)
**Tags:** `+web-app` `+mcp-client` `+ui`

---

As a Law-Corp user,
I want to browse available MCP prompt templates, fill in their arguments, and see the generated prompt,
So that I can use the firm's reusable prompt workflows through the web app.

## Implementation Steps

- [ ] Create a `Prompts.razor` page at `/prompts`
- [ ] Call `IMcpClientService.ListPromptsAsync()` on page load and display prompts in a list
- [ ] Each prompt card shows: name, description, argument definitions
- [ ] Clicking a prompt opens a form for its arguments (similar to tool parameters in 8.2.3)
- [ ] "Generate" button calls `IMcpClientService.GetPromptAsync(promptName, arguments)`
- [ ] Display the resulting prompt messages in a chat-like or structured view

## Acceptance Criteria

- [ ] All prompts returned by the MCP server are listed
- [ ] Argument forms adapt to each prompt's definition
- [ ] Generated prompt messages are displayed correctly
- [ ] The UI handles prompts with no arguments (direct invocation)
